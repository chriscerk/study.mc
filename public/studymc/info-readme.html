StudyMC

Dev. ReadMe


StudyMC (Study Medicinal Chemistry) is an Angular 2 application.


Installation

Dependencies

`composer`, `nodejs` & `typescript`

Environment Setup

Laravel Homestead: https://www.laravel.com/docs/5.3/homestead

Running the Environment

  1. `cd Homestead`
  2. `vagrant up`
  3. Go to http://study.mc/ in your browser. (Or the specific domain setup for Homestead at C:\Windows\System32\drivers\etc\hosts)

Building

  1. `cd studymc`
  2. `gulp`
